# 🏋️ Gym Pose Detection System

A real-time AI-powered gym pose detection system built using MediaPipe and OpenCV.  
This project tracks human body movements through a webcam and analyzes exercise form to help users perform workouts correctly and avoid injuries.

---

## 🚀 Features

- 🎥 Real-time pose detection
- 🧍 Full body landmark tracking
- 📏 Joint angle calculation
- 🔢 Exercise repetition counting
- ⚠️ Form correction feedback
- ⚡ Fast and lightweight

---

## 🛠️ Tech Stack

- Python 3.x
- OpenCV (opencv-python)
- MediaPipe
